# EditorialDesk Team (Personas + Avatar Directions)

## Global art direction (apply to every avatar)

* **Format:** head-and-shoulders portrait, centered, consistent framing across all characters
* **Style:** warm semi-realistic illustration, soft studio lighting, subtle paper grain texture
* **Palette:** warm beige background, terracotta accents, soft gray neutrals; muted sage allowed for one role
* **Backgrounds:** minimal, soft-focus, *no readable text*, only faint icons/shapes
* **Clothing:** modern professional-casual, varied silhouettes, no heavy patterns
* **Diversity rule:** each person must be **visibly distinct** (different age group, ethnicity, face shape, hairstyle). No “same face” reuse.

## Consistency checklist (before generating)

* No readable text anywhere (only lines, blocks, checkmarks).
* Each character: different age + ethnicity + face shape + hairstyle.
* Same framing, same lighting, same warm pastel palette.
* Role readability: prop + badge/icon + outfit silhouette must communicate the job.
